## Design Context

### Users
- Primary audience balance: equal priority between borrowers/public users and librarian/admin staff.
- Context of use: users need to complete common library workflows quickly (discover books, manage accounts, circulation/admin tasks) without confusion.
- Core job to be done: make the site easy to navigate, easy to understand, practical for daily tasks, and pleasant to use.

### Brand Personality
- Three-word personality: minimal, aesthetic, calm.
- Emotional goals: trust, calm, and confidence.
- Voice and tone: clear, straightforward, and reassuring rather than promotional or flashy.

### Aesthetic Direction
- Direction: light-first, editorial-minimal interface with calm visual rhythm.
- Existing visual foundation to preserve: warm paper palette, serif + sans pairing, restrained accents.
- Reference influence: Notion-like color discipline (neutral, practical, low-noise).
- Anti-reference: avoid overcomplicated layouts and text-heavy, cluttered screens.
- Theme: light mode only for now.

### Design Principles
1. Clarity before decoration: each screen should make the next action obvious within seconds.
2. Practical calm: keep layouts visually quiet, task-focused, and free from unnecessary UI noise.
3. Equal-role usability: borrower and staff surfaces should feel equally understandable and efficient.
4. Progressive disclosure: show essential information first; reveal details only when needed.
5. Consistent trust cues: maintain stable typography, spacing, and feedback patterns to reinforce reliability.
